How to play:
Use your Arrow keys, WASD keys and/or HJKL keys to move the tiles. When two tiles with the same number touch, they merge into one! Use the R key to reset.

Note: This site is a forked version of the original 2048. It's to demonstrate an AI written for the 2048 game. You can play the game and use the AI for help.
I also made a chrome extension using this AI. If you are using chrome, you can install the extension and go to the original 2048 website to play with it.
